# 2A1B Game Guide

## What is 2A1B?

2A1B is an alphabet-themed variant of the classic 2048 puzzle game. Instead of numbers, you merge letter tiles from A through Z.

## How to Play

1. Use arrow keys (or swipe on mobile) to slide all tiles in one direction
2. When two tiles with the same letter touch, they merge into the next letter (A+A=B, B+B=C, etc.)
3. A new tile spawns after each move (A or B)
4. Goal: Reach the letter K (or as high as possible!)
5. Game over when no more moves are possible

## Scoring

- Each merge earns points based on the tile value
- Higher letter merges = more points
- Try to chain merges for maximum score

## Strategy Tips

- Keep your highest tile in a corner
- Build a snake pattern to set up chain merges
- Avoid spreading tiles — keep them consolidated
